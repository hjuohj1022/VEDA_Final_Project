# STM32 Motor Control Command Reference

본 문서는 ESP32(MQTT 클라이언트)에서 STM32(SPI 슬레이브)로 전송하는 모터 제어 명령어 형식을 정의합니다. 모든 명령어는 문자열 형태이며, 최대 30바이트를 초과하지 않아야 합니다.

## 1. 기본 명령어 형식
`motor<ID> <Action> <Value/Type>`

- **ID**: 모터 번호 (1, 2, 3)
- **Action**: 동작 종류 (left, right, set, release, stop)
- **Value/Type**: 각도 값 또는 동작 타입 (press, release, 숫자)

---

## 2. 명령어 상세

### 2.1. 연속 이동 (GUI 버튼 방식)
버튼을 누르고 있는 동안 모터가 1도씩 계속 이동하고, 떼면 정지하는 방식입니다.

| 명령어 | 설명 | 비고 |
| :--- | :--- | :--- |
| `motor1 left press` | 모터 1번을 왼쪽(CCW)으로 연속 이동 시작 | `Motor_Update()`에서 처리 |
| `motor1 right press` | 모터 1번을 오른쪽(CW)으로 연속 이동 시작 | `Motor_Update()`에서 처리 |
| `motor1 release` | 모터 1번 이동 정지 | 버튼을 뗐을 때 전송 |

### 2.2. 절대 각도 이동
특정 각도로 즉시 이동합니다.

| 명령어 | 설명 | 범위 |
| :--- | :--- | :--- |
| `motor1 set 90` | 모터 1번을 90도(중앙)로 이동 | 0 ~ 180도 |
| `motor2 set 0` | 모터 2번을 0도로 이동 | |
| `motor3 set 180` | 모터 3번을 180도로 이동 | |

### 2.3. 상대 각도 이동
현재 위치에서 지정한 각도만큼 더 이동합니다.

| 명령어 | 설명 | 비고 |
| :--- | :--- | :--- |
| `motor1 left 10` | 현재 위치에서 왼쪽으로 10도 더 이동 | 상대 이동 |
| `motor2 right 20` | 현재 위치에서 오른쪽으로 20도 더 이동 | 상대 이동 |

### 2.4. 기타 정지 명령
연속 이동 중인 모터를 강제로 정지시킵니다.

| 명령어 | 설명 |
| :--- | :--- |
| `motor1 stop` | 모터 1번 정지 (`release`와 동일 동작) |

---

## 3. 주의 사항
1. **대소문자 미구분**: `MOTOR1 SET 90`과 `motor1 set 90`은 동일하게 처리됩니다.
2. **공백 처리**: 명령어 시작 부분이나 단어 사이의 추가 공백은 자동으로 무시됩니다.
3. **각도 제한**: 하드웨어 설정(`motor.h`)에 따라 0~180도 범위로 소프트웨어 제한이 걸려 있습니다.
4. **파싱 실패**: 형식이 맞지 않는 명령어가 들어오면 STM32는 `Unknown cmd` 로그를 출력하고 명령을 무시합니다.
